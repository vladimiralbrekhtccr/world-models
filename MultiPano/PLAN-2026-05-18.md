## Things to NOT do (gotchas learned today)

- **Don't install ffmpeg via `conda install -c conda-forge ffmpeg`** —
  conda resolves the deps and brings GraalPy, which replaces CPython
  and breaks numpy/torch imports. Use `imageio-ffmpeg` or `static-ffmpeg`
  via pip instead.
- **Don't try to install tinycudann** for splatfacto. It's not needed
  (splatfacto uses gsplat). Build fails anyway because system gcc=12 >
  CUDA 11.8's gcc≤11 limit.
- **Don't trust `pip install -U setuptools`** for native builds —
  setuptools≥70 drops `pkg_resources` which old build hooks need.
  Pin `setuptools<70`.
- **Don't use sequential-only COLMAP matching** on phone video with
  scene jumps; use exhaustive. And use mpdecimate when extracting
  frames — drops near-duplicate frames automatically (nerfstudio
  does this internally via `ns-process-data`).
